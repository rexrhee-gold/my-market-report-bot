# 장전 최종 체크 — 오전 8:40 기준

## 분석 기준

오전 7:40 리포트를 기본 시나리오로 사용한다.

추가로 오전 8:40 기준 최신 정보를 확인한다.

반드시 확인할 항목:

- KOSPI200 선물 또는 야간선물 흐름
- 미국 선물 흐름
- 원/달러 환율 또는 NDF 환율
- 달러 인덱스
- 미국 10년물 금리
- WTI, Brent 유가
- 일본·중국·홍콩 등 아시아 주요 지수 선물 또는 개장 전 분위기
- 국내 장전 주요 뉴스
- 반도체, 2차전지, 금융, 조선, 방산, 바이오 관련 장전 뉴스
- 공시, 실적, 정책, 지정학 이슈
- 외국인 수급에 영향을 줄 수 있는 변수

데이터가 명확하지 않은 항목은 추정하지 말고 확인 필요라고 표시한다.

## 작성 원칙

8:40 리포트는 길게 설명하지 않는다.

개장 직전 실제 매매 판단에 필요한 내용만 압축한다.

반드시 다음 관점을 포함한다.

- 7:40 리포트 대비 바뀐 점
- 갭상승 가능성
- 갭하락 가능성
- 장초반 추격매수 가능 여부
- 관망 필요 여부
- 오늘 우선순위 섹터
- 오늘 피해야 할 섹터
- 9:00~9:30 실행 전략
